### 添加到应用

1. 创建或编辑聊天流或工作流应用
2. 在工具选择面板中，选择"DOC"工具
3. 根据需要在应用流程中配置工具
4. 保存并发布您的应用


## 使用方法

您可以使用此工具将 Markdown 内容转换为 DOCX 格式，需要两个参数：

- **markdown_content**（必填）：您要转换的 Markdown 内容

- **title**（可选）：文档的标题（默认为"Document"）
