### アプリケーションへの追加

1. チャットフローまたはワークフローアプリケーションを作成または編集します
2. ツール選択パネルで「DOC」ツールを選択します
3. 必要に応じてアプリケーションフローでツールを設定します
4. アプリケーションを保存して公開します


## 使用方法

このツールを使用して、Markdown コンテンツを DOCX 形式に変換できます。2つのパラメータが必要です：

- **markdown_content**（必須）：変換したい Markdown コンテンツ

- **title**（オプション）：ドキュメントのタイトル（デフォルトは「Document」）
